# What is the TestFlight version of Tachimanga

TestFlight version is the beta version of Tachimanga, which includes new features, bug fixes, and weekly updates. It allows you to access all the new features earlier.

# How to get a TestFlight slot

If you have subscribed to the premium, you can get a reserved TestFlight slot during the subscription period, granting you early access to all new features.

You can request your TestFlight invite using the form below.

# Request TestFlight Invite

Please refer to the guide below to find your Order ID
The email does not need to match your Apple ID email. However, you need to click the link in the invitation email on the device where you are logged in with the Apple ID used for purchase.

# How to find your Apple Order ID

To find your Apple Order ID, follow these steps:

  1. Open the App Store app.
  2. Tap the your photo at the top of the screen.
  3. Tap Purchase History to see your purchases. You might be asked to sign in.
  4. Find Tachimanga in the list, tap it to open the details page, and you'll see your Apple Order ID there.

You can also find the order id in the in-app purchase receipt email Apple sends you.
If you can't find it, please watch the video tutorial (opens new window).

# FAQ

# What should I do if TestFlight asks for a Redeem code?

If TestFlight asks for a Redeem code, simply close TestFlight and click on the TestFlight link again.

# What can I do if the TestFlight one has expired?

Don't worry, each update extends the expiration for 90 days. Even if it expires, you can simply download the App Store version to replace it, and all your data will be preserved.

# Will installing the TestFlight one result in any data loss?

No, there won't be any data loss when installing the TestFlight one.

# How many slots can one order get?

The slot is bound to the Apple ID that accepts the invite. One order can only get the invite link once. If you haven't received the invite link within 15 minutes, you can try with a different email. However, one order can only change the email once.
