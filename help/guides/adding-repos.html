# Getting Started

Tachimanga is a full-featured reader that supports reading from local source and external third-party extensions.

# Reading from Local source.

Read content stored locally on your device.
See the Local source guide (opens new window) for instructions.

# Reading from External third-party extensions.

External third-party extensions add additional sources to Tachimanga.
Tachimanga supports most of Mihon/Tachiyomi extensions, so you can add Mihon/Tachiyomi extensions repositories by going to More -> Extensions -> Extension repositories.

TIP

No extensions/sources out of the box. Feel free to use Google or other resources to find what you need.

TIP

These extensions are not developed or maintained by the Tachimanga developers. Therefore, we do not provide any info regarding those extensions.
