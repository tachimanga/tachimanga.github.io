# Repositories

# What are extension repositories?

Extension repositories are online platforms where users can find and install extensions for the Tachimanga. These extensions enhance Tachimanga's functionality by accessing to different manga websites or integration with external services such as Komga.

# How to find extension repositories?

There are several methods:

  1. Search for "Tachiyomi extensions" on GitHub (opens new window).
  2. Search for "Source extensions for the Tachiyomi app" on Google (opens new window).

If you haven't found what you needed, feel free to join the Discord (opens new window) for support.
