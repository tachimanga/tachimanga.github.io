# Will the Premium subscription work across multiple devices?

Yes. Please follow these steps:

  1. Install Tachimanga on another device using the SAME Apple ID.
  2. If Premium is not active, go to the Get Premium page and tap Restore Purchase.

Since v4.11, Premium is usually activated automatically on new devices, so this step may not be necessary.
