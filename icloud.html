# How to enable iCloud drive?

# 1. Please turn on iCloud Drive in the system settings.

# 2. Please turn on Tachimanga in apps using iCloud.

# 3. Please turn on iCloud Drive in the Files app.

If iCloud is still not available, please try restarting your phone. If that doesn't work, please join the Discord (opens new window) for support.
